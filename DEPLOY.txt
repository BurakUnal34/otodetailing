Yerel geliştirme
================
- DATABASE_URL="file:./dev.db" (SQLite, prisma/dev.db)
- İlk kurulum: npx prisma db push && npm run db:seed

Canlı (Vercel vb.)
==================
- PostgreSQL sağlayın (Neon, Supabase…). prisma/schema.prisma içinde provider = "postgresql" yapın.
- prisma migrate ile şema sürümleyin; build öncesi migrate deploy ekleyin.
- BLOB_READ_WRITE_TOKEN (Vercel Blob) önerilir; yoksa kalıcı disk gerekir.
- NEXTAUTH_URL / NEXT_PUBLIC_SITE_URL gerçek https adresiniz olsun.

Docker ile yerel Postgres (isteğe bağlı)
==========================================
- Docker Desktop kuruluysa: docker compose up -d
- .env DATABASE_URL postgresql://otodetailing:otodetailing@localhost:5432/otodetailing?schema=public
- schema.prisma provider postgresql olmalı.
